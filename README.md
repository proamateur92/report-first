# 항해 99 리액트 1주차 과제
- React hook을 사용하여 요일 별로 어떻게 보냈는지에 대한 점수를 부여한다.
- state값을 통해 평점을 계산하였다.
- React Router를 통해 요일마다 상세페이지로 이동하도록 하였다.
- 상세페이지에서는 각 요일에 대해 평점을 매길 수 있다.
- mouseEnter Event를 통해 평점이 변경되도록 구현하였다.
